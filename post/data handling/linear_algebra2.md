# linear algebra (선형대수) 2

## Variance (분산)

+ 분산은 데이터가 **얼마나 퍼져있는지** 측정하는 방법.  
+ 각 값들의 평균으로부터 차이의 제곱의 평균

**분산 계산시의 자유도** (ddof)
+ 샘플의 분산 : n-1
+ 모집단의 분산 : 0
  
## Standard Deviation (표준편차)

표준편차는 분산의 값에 루트를 씌운것  $\sqrt()$  

## Covariance (공분산)

1개의 변수의 값이 변화할 때 다른 변수가 어떠한 **연관성을 나타내며 변화하는지**를 측정하는 것.
